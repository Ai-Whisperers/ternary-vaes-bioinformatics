# CLAUDE PROJECT ENTRY POINT

**Project**: Ternary VAEs Bioinformatics
**Stack**: Python 3.8+, PyTorch 2.0+, Hyperbolic Geometry

## Quick Start

- **Install**: `pip install -r requirements.txt`
- **Test**: `/test` (or `pytest`)
- **Run**: `python src/main.py` (Verify entry point)

## Key Directories

- `src/`: Core VAE implementation.
- `DOCUMENTATION/`: Knowledge base.
- `scripts/`: Experiments.

## Rules

- Follow PEP 8.
- Use Type Hints.
