# ATOM3D: Tasks On Molecules in 3 Dimensions

**Author:** Townshend, R.J.L., et al.
**Year:** 2020
**Link:** [arXiv](https://arxiv.org/abs/2012.04035)

## Abstract (Summary)

A benchmark dataset and framework for 3D molecular learning. It established standard tasks (like ligand binding, structure prediction) to evaluate geometric deep learning models, facilitating progress in the field.

## Key Concepts

- 3D Molecular Benchmarks
- Geometric Deep Learning Tasks
- Structural Biology Datasets
