## Using Diagrams

### Viewing
Mermaid diagrams (`.mmd` files) render in:
- GitHub (native support)
- VS Code (with Mermaid extension)
- Most modern documentation tools

### Embedding
Include in markdown:
```markdown
```mermaid
graph LR
    A --> B
```
```

See [DIAGRAM_INVENTORY.md](DIAGRAM_INVENTORY.md) for complete listing.

## Creating New Diagrams

1. Use `.mmd` extension for Mermaid files
2. Place in appropriate subsection
3. Add to DIAGRAM_INVENTORY.md
