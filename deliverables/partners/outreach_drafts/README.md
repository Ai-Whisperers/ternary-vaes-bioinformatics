## Email Format (All Drafts)

All 100 drafts follow these standards:

- **Under 150 words** (most under 130)
- **No tables** in email body
- **Specific subject lines** referencing their work
- **One clear ask** per email
- **Team signature:**
  ```
  Best,
  Ivan Weiss Van Der Pol
  Jonathan Verdun
  Kyrian Weiss Van Der Pol
  AI Whisperers | CONACYT Paraguay
  github.com/Ai-Whisperers/ternary-vaes-bioinformatics
  ```

## Sending Workflow

1. **Review draft** - Check personalization is accurate
2. **Verify email** - Double-check address spelling
3. **Check timing** - Best to send Tue-Thu, morning their timezone
4. **Send** - Copy email content from draft
5. **Log** - Update DRAFT_TRACKER.md with sent date
6. **Follow up** - Set reminder for 1 week if no response
